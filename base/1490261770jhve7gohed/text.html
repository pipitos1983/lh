Чтобы заходить локально любым пользователем:


в файле

/etc/postgresql/9.4/main/pg_hba.conf

строку

local all postgres peer

привести к виду

local all postgres trust



В стандартном сценарии используется локальный пользователь postgres:


#su postgres



Смена пароля:


$psql -U postgres -d template1 -c "ALTER USER postgres PASSWORD 'password'"
