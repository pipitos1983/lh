aptitude install redmine redmine-pgsql


На сервере СУБД постгрес:


CREATE ROLE redmine LOGIN ENCRYPTED PASSWORD 'redmine' NOINHERIT VALID UNTIL 'infinity';

CREATE DATABASE redmine WITH ENCODING='UTF8' OWNER=redmine;


сконфигурировать редмайн для работы с субд:


cp /usr/share/redmine/templates/database.yml.template /etc/redmine/default/database.yml


содержимое:


production:

adapter: postgresql

database: redmine

host: srvr-db

username: redmine

password: redmine

encoding: utf8


Начальное заполнение БД:


cd /usr/share/redmine

RAILS_ENV=production rake db:migrate


убираем прочие настройки


rm /etc/apache2/sites-enabled/000-default.conf
