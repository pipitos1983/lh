назначение пароля через локальную консоль:

mysqladmin -u root password '123'


создание пользователя:


CREATE USER 'newuser'@'localhost' IDENTIFIED BY 'password';


или для возможности доступа по сети

CREATE USER 'newuser'@'*' IDENTIFIED BY 'password';



Важно не забыть что можно добавить оба таких "пользователя" и выдавать доступ на конкретные базы в зависимости от потребностей.

(к одной только локальный доступ к другой по сети)


FLUSH PRIVILEGES;
