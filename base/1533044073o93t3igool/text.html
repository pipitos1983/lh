монтирование шары средствами ос:

1 создаем каталог "testdir" в рамках пользователя "test" на облаке



2 монтируем в него файловую систему средствами ос

mount -t cifs //192.168.1.3/volume_1 /var/www/cloud/html/data/test/testdir rw,username=smbuser,password=******,iocharset=utf8


3 сканируем файлы выбранного пользователя

sudo -u www-data php occ files:scan test
