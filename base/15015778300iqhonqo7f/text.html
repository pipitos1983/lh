По умолчанию при установке выключен сетевой доступ.

Можно включить в mmc консоли управления "Sql Server Configuration Manager".

Раздел "Сетевая конфигурация SQL Server" -> протоколы для [имясервера] -> TCP/IP


PS:

новые базы создаются с простой схемой восстановления и шагом роста 64Мб так что править уже не нужно.
