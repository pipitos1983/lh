1 скачать с офсайта нужный rpm:

wget -c http://ftp.otrs.org/pub/otrs/RPMS/rhel/7/otrs-5.0.20-01.noarch.rpm



2 установить скачанный пакет с учетом зависимостей

yum localinstall otrs-5.0.20-01.noarch.rpm


у меня было 138 зависимостей, но всего что нужно все равно не доставилось.



3 Установка недостающего

yum install postgresql perl-DBD-Pg



4 Включение сервисов:

systemctl enable httpd

systemctl start httpd


service postgresql initdb

systemctl enable postgresql

systemctl start postgresql



5 правим /var/lib/pgsql/data/pg_hba.conf:


# IPv4 local connections:

host all all 127.0.0.1/32 md5

(меняем ident на md5 в конце)



6. Создаем пользователя и базу:


# su postgres

psql


postgres=# CREATE DATABASE otrsdb;

CREATE DATABASE

postgres=# CREATE USER otrsdbuser WITH password 'otrsdbpassword';

CREATE ROLE

postgres=# GRANT ALL ON DATABASE otrsdb TO otrsdbuser;

GRANT

postgres=# \q




7. Завершаем установку через веб. (http://server.address/otrs/installer.pl)
