Работа вставленного ключа обеспечивается 2 пакетами - сервис, и модуль ядра.


debian8{


wget http://download.etersoft.ru/pub/Etersoft/HASP/stable/x86_64/Debian/8/haspd_7.40-eter10debian_amd64.deb

wget http://download.etersoft.ru/pub/Etersoft/HASP/stable/x86_64/Debian/8/haspd-modules_7.40-eter10debian_amd64.deb


aptitude install libc6-i386


dpkg -i haspd_7.40-eter10debian_amd64.deb

dpkg -i haspd-modules_7.40-eter10debian_amd64.deb


}


centos7{

yum -y install glibc.i686

rpm -Uvh http://download.etersoft.ru/pub/Etersoft/HASP/stable/x86_64/CentOS/7/haspd-7.40-eter10centos.x86_64.rpm

rpm -Uvh http://download.etersoft.ru/pub/Etersoft/HASP/stable/x86_64/CentOS/7/haspd-modules-7.40-eter10centos.x86_64.rpm

}


Настройка сервиса:


/etc/haspd/hasplm.conf

NHS_IP_LIMIT = 127.0.0.1, 10.200.0.0/24


Управление стандартное:


service haspd start

service haspd status
