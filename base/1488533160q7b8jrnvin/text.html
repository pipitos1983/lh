"Apache2 в ОС Astra Linux Special Edition работает только с принудительной аутентификацией".


aptitude install apache2 libapache2-mod-auth-pam


в настройки хоста добавить:

<Directory /var/www/>

AuthPAM_Enabled on

AuthType Basic

AuthName "PAM authentication"

require valid-user

....


a2enmod auth_pam

service apache2 reload
