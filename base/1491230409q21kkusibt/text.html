1. Проверить sysctl-переменную vfs.zfs.vol.mode (Должен быть 1):


sysctl -a | grep zfs.vol.mode

vfs.zfs.vol.mode: 2


Если другое то установить в 1 (System->Tunables)


2. Создать Volume:


zfs create -V 16G -o org.freebsd:swap=on -o volblocksize=4K -o checksum=off -o compression=off -o dedup=off -o sync=disabled -o primarycache=metadata zstor/swap0


3. Включить swap или перегрузить систему:


swapon /dev/zvol/zstor/swap0
