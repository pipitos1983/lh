Гостевая венда определяет что находится в виртуальном окружении и не дает использовать (и даже ставить) hyper-v.


убираем из инвентори виртуалку


редиктируем vmx файл добавляя строки:
vhv.enable = "TRUE"

hypervisor.cpuid.v0 = "FALSE"


добавляем обратно виртуалку.
