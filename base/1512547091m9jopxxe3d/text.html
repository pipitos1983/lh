gpart - утилита для управления разделами.

gpart show - просмотр конфигурации всех дисков.

gpart destroy -F /dev/da1 - удаление таблицы и всех разделов.

(без ключа -F таблица удалена не будет если остались разделы)

gpart create -s GPT /dev/da1 - создание таблицы разделов.

gpart delete -i 2 da1 - удаление раздела (2 раздел на диске da1)
