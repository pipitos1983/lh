каталог в каталог (например для chroot ):


mount --bind /dev /mnt/dev

mount --bind /sys /mnt/sys

mount --bind /proc /mnt/proc


Образ файловой системы в каталог:


mount -o loop somefile.img /mnt/image

mount -o loop somefile.iso /mnt/cd-image


в старых системах бывает нужно указать ФС:


mount -o loop -t iso9660 somefile.iso /mnt/cd-image

mount -o loop -t ext3 somefile.img /mnt/image


монтирование подкачки:


запись в /etc/fstab:

/swap.img none swap sw 0 0


если все верно прописано в /etc/fstab:


swapon -a

(отключение swapoff -a)


Как подключить указывая конкретный файл без fstab я не знаю.

Устройства без fstab можно подключить используя ключ -L.

Список аргументов для него см. в /proc/partitions.
